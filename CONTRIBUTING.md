# Contributing

## Accepted

- Bug fixes
- Security fixes
- Documentation fixes

## Not Accepted

- New features
- Bundlers
- Test frameworks
- npm publishing

## Commits

Use conventional commits: `type(scope): description`

```
fix(quality): handle zero-file PRs
docs(readme): update setup instructions
```
